# AI Innovators Summit 2025

Simple steps to run this on any laptop.

## Prerequisites
- Node.js 18+ (install from https://nodejs.org)
- Git

## Clone and run
```bash
git clone https://github.com/your-org/ai-innovators-summit-2025.git
cd ai-innovators-summit-2025
npm install
```
Open the URL shown by Vite (defaults to http://localhost:3000).

## Build and Docker (optional)
- Build: `npm run build`
- Docker: `docker build -t ai-innovators-summit . && docker run -p 3000:80 ai-innovators-summit`
